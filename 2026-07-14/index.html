Generalize LMMs to Versatile Visual Modalities via Fabricated Modality Synthesis

Shihao Yuan, Yuanze Li, Ruyi Zhang, Ming Liu, Wangmeng Zuo · Harbin Institute of Technology
ECCV 2026 acceptedarXiv 2607.10308v1Announced 2026-07-11

VVM-Tuning asks whether an LMM can handle thermal, depth, X-ray, optical-flow-like, and synthetic modalities without in-modality training.

Source: arXiv:2607.10308v1 · §cover p.1 · §1 pp.1-4
TL;DR

Do not teach the model every sensor. Teach it to separate what the scene is from how this sensor paints it.

The paper treats non-RGB modalities as different samplings of the same physical world: shared semantics stay stable; appearances and color meanings change.

Abstract p.1; Fig.1 p.2
Problem

Current LMMs are mostly RGB-native, even when they look multimodal

  • Non-RGB ability often comes from seen data already included in scaling: DocQA, ScienceQA, RGB-D / RGB-T, domain-specific medical or remote-sensing datasets.
  • ImageBind / PandaGPT-style transfer still relies on paired or seen modality encoders, so it does not answer the unseen visual-modality question.
  • The paper makes the question explicit: can LMMs generalize to versatile visual modalities without training on in-modality data?
The gap is not “more benchmarks”; it is an alignment problem between semantics, appearance, and physical meaning.
§1 pp.1-2; §2 p.5
Core observation

A thermal image and an RGB image can share a scene while disagreeing on appearance

  • Modality-unaware perception: recognize shape, edge, contour, position, and scene semantics even without sensor knowledge.
  • Modality-aware understanding: use context to map a color or pattern back to temperature, depth, radiation, flow, etc.
Fig.1 p.2; §1 pp.2-3
Human analogy

Before knowing the sensor, we can still perceive basic visual elements

  • This motivates a two-level view of visual perception.
  • Semantic perception: objects and scene relations.
  • Photographic perception: visible clues such as color, lines, textures, and shapes before physical interpretation.
The training target is not “understand thermal immediately”; it is “keep visual clues disentangled enough that context can later interpret them.”
Fig.2 p.3; §1 p.3
Contribution map

Three artifacts: synthesis pipeline, tuning tasks, and VVM-Bench

  1. Fabricated modality synthesis: RGB scenes are transformed into diverse pseudo-modal images and paired with synthetic modality contexts.
  2. VVM-Tuning: instruction tuning separates modality-unaware perception from modality-aware understanding.
  3. VVM-Bench: 6 modalities and 11 VQA tasks test perception and context-grounded understanding.

Method 1 / Modality synthesis

Fabricate appearances first, then fabricate the words that explain them

  • Image branch: RGB to grayscale, histogram spline randomization, 22 colormaps, noise / smoothing / blur.
  • Context branch: masks or object labels connect colors to objects and physical attributes; if no annotation exists, an LMM writes context from examples.
Fig.3 p.6; §3.2 pp.6-7
Method 1 / Appearance diversity

The synthetic images are deliberately simple, but cover many “sensor-like” looks

  • Spline S(x) remaps grayscale pixel values while preserving valid [0,255] range.
  • Pseudo-color maps mimic scientific visualization and sensor outputs.
  • Augmentations simulate noisy or low-resolution imaging processes.
I_hat = S(I), where S: [0,255] to [0,255] is a rejected-if-invalid cubic spline
The point is not photorealism; it is pressure-testing whether semantics survive appearance changes.
§3.2 p.7; Appendix C.1 pp.28-29; Alg.A1 p.29
Method 2 / Task construction

VVM-Tuning trains two abilities instead of one entangled captioning skill

  • Perception tasks: semantic description, semantic QA, photographic description, photographic QA.
  • Understanding tasks: modality description and modality QA built from semantic + photographic descriptions plus context.
Fig.5 p.8; §3.3-§3.4 pp.8-9
Method 3 / Objective

The learning algorithm is deliberately ordinary SFT

LSFT = -Σi=1..n log pitheta(yi | X, y<i)
  • All structure is in the data and task design, not a new model architecture.
  • Training optimizes generated answer Ta toward ground-truth Tgt.
  • This makes the result relevant to many base LMM families: LLaVA, Qwen2.5-VL, Qwen3-VL.
A simple objective makes the ablation cleaner: gains are more attributable to task decomposition and synthetic modality construction.
§3.1 p.6; §4.1 pp.10-11

Why it works

VVM-Tuning weakens over-alignment to RGB appearance

  • The same semantics appear through many fabricated appearances, so the model is trained to recover scene meaning without relying on RGB-like color statistics.
  • Photographic tasks prevent the model from ignoring visual details; it must still name colors, edges, shapes, and relative sizes in the fabricated image.
  • Modality contexts then map those details to physical attributes such as temperature, depth, motion direction, hardness, humidity, or radiation.
This is closer to “latent factor disentanglement by instruction data” than to ordinary domain fine-tuning.
§3.3-§3.4 pp.8-9; §4.4 pp.13-14
What is fragile

Synthetic modalities teach perception better than they teach real physical meaning

  • Understanding gains are consistently lower than perception gains: +3.8% vs +8.2% average.
  • The authors explicitly identify a synthetic gap: made-up physical meanings are harder to imitate than visual appearance.
  • Multiple-choice VQA is convenient, but it can hide shallow option-matching and does not fully evaluate multi-hop modality reasoning.
The method is a strong perceptual basis, not a complete replacement for domain-specific sensing knowledge.
§4.3 p.13; §A.5 p.25
Engineer’s read

The useful pattern is data factorization, not fancy architecture

  1. Hold semantics fixed: start from paired RGB/fabricated views of the same scene.
  2. Vary appearance aggressively: random splines, colormaps, blur/noise, style transfer, flow visualization.
  3. Ask two question families: one that ignores modality meaning, one that requires textual modality context.
  4. Measure separately: perception gain and context-understanding gain should not be collapsed into one score.
§3.2-§3.5 pp.6-10; §4 pp.10-14

Appendix · Limitation in the authors’ words

“Our tuning only disentangles that alignment by semantics and photographics, not executing a completely new alignment or injecting new knowledge.”

這是把語意與 photographic perception 解耦，不是建立全新 alignment，也不是把新領域知識灌進模型。§A.5 p.25

  • Generalization remains bounded by the base LMM’s known objects and abilities.
  • Non-RGB downstream tasks often require specialized domain knowledge.
  • Choice-question VQA can be guessed from options, especially when context is strong.
§A.5 p.25

Takeaway

VVM-Tuning is a clean reminder: for multimodal models, “more modalities” can sometimes start from better factorization of one modality.
