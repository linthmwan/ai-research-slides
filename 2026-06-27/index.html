Resp-Agent

Pengfei Zhang · Tianxin Xie · Minghao Yang · Li Liu
The Hong Kong University of Science and Technology (Guangzhou)
ICLR 2026arXiv:2602.15909v3AI Agent / Multimodal System

今日一句話

把 failure mining 接上可控生成器，augmentation 就從「補資料」變成「agent 控制迴路」。

Thinker-A2CA 找診斷弱點，Generator 合成目標病例，Diagnoser 用 text-audio anchors 捕捉短暫病理聲音。

§1 p.2; §4 p.4; §6 p.10

問題不是只有「分類器不夠強」

  • Representation gap: spectrogram / CNN pipeline 會模糊 crackles 等 transient events；text-only EHR 又缺少客觀聲學證據。
  • Data gap: 公開呼吸聲資料集小、異質、長尾，且很少有標準化 multimodal clinical context。
  • System gap: 診斷與生成通常分開做，無法把「模型錯在哪」直接轉成「該合成什麼」。
  • Evaluation gap: in-domain split 容易樂觀；paper 強制 source-disjoint Test-CD。
Resp-Agent 的主張：用 agent planner 把資料、生成、診斷三者綁成閉環。
§1 p.1–2; §3 p.3–4

核心貢獻：一個 benchmark + 一個閉環 agent system

  1. Resp-229k: 229,101 clips / 408 h / 16 classes，附 LLM-rendered clinical summaries。
  2. Thinker-A2CA: compute-aware LLM planner，依 rationales、error profiles、confidence 安排 targeted synthesis。
  3. Generator: Qwen3-0.6B-Base + modality injection + CFM，解耦 pathology content 與 acoustic style。
  4. Diagnoser: Longformer modality weaving + sparse audio anchors，把 EHR text 對齊 10 秒音訊中的 transient cues。
  5. Evidence: ICBHI Score 72.70；Test-CD Macro-F1 從 0.212 拉到 0.598。
§3 p.3–4; §4 p.4–8; §5 p.8–10

Generator Stage 1：把 text-only LLM retool 成 acoustic unit generator

Figure 2 p.5; §4.1.1 p.4–6

Modality injection：content 與 style 被刻意拆開

P = PoolK(Z),   Estyle = StyleProj(P)
Prompt = [DIAGNOSIS] d + [AUDIO0] ... [AUDIOK-1]
  • Z 是 10 s / 16 kHz reference audio 的 BEATs framewise features；Pool 後得到 K 個 style descriptors。
  • [AUDIO_i] token 的 embedding 由 Estyle row 替換，但 language backbone 架構不改。
  • Resp-MLLM autoregressively predicts BEATs codebook discrete acoustic units。
  • 約 10% preceding unit embeddings 被 [BEATs MASK] 取代，避免看見 oracle t-1 token。
§4.1.1 p.5–6; Eq.1–3

Generator Stage 2：CFM 讓 discrete units 變成可聽 waveform

xt = (1-t)x0 + tx1
dxt/dt = vtheta(xt, c)
  • DiT-style conditional flow matching decoder reconstructs mel-spectrograms。
  • Vocos 將 mel 轉回 waveform。

dual-path condition c

  • Content stream: predicted unit indices embedded and interpolated to mel frame rate。
  • Timbre stream: time-averaged BEATs features broadcast across time。
  • Reference prefix: short ground-truth mel prefix encourages accurate continuation。
工程上很乾淨：LLM 決定「要什麼病理內容」，CFM 決定「怎麼像那個聲音」。
§4.1.2 p.6; Eq.4–5

Diagnoser：把 EHR token 與 audio token 放進同一條 Longformer sequence

Figure 3 p.7; §4.2 p.6–8

Audio anchors：sub-100ms 的 cross-modal routing grid

T = 496 frames / 10 s ⇒ hop Δt = 20.16 ms
stride s = 4 ⇒ anchor spacing sΔt = 80.6 ms, worst-case snap ≤ 40.3 ms
  • Global set G = {[CLS], [DESCRIPTION]} ∪ A；anchors A = {a1+ks}。
  • G tokens mutual-attend to entire sequence；其他 token 仍維持 local sliding window。
  • 文字症狀如 “wheeze” 可以直接 query 遠處短暫 acoustic event，不必等 local window 慢慢傳訊。
  • 這是 paper 的 mechanism 皇冠：低成本但有臨床時間尺度的注意力路由。
§4.2.2 p.7–8
Mechanism crown

Anchors 決定模型「看得到哪裡」；Generator 決定訓練時「看過什麼 rare cases」。

Thinker-A2CA 的價值，是把這兩件事用 failure signal 連成同一個 active curriculum。

Negative controls：傳統 imbalance remedy 不夠

  • Class-weighted CE：Macro-F1 0.248；Focal Loss：0.267；都遠低於 CE + Thinker-A2CA 的 0.598。
  • Naive augmentation（duplication、pitch/time shift、noise）在 cross-domain test 讓 audio-only Conformer Macro-F1 下降 0.0247。
  • 原因很合理：病理聲音的 low-energy micro-structures 很容易被 pathology-agnostic transform 破壞。
  • 所以 paper 的關鍵不是「平衡類別數」，而是「合成有診斷資訊的 rare/hard cases」。
如果要借鑑到其他醫療模態，第一步不是加 loss weight，而是建立 failure-aware controllable generator。
§5 p.9; Appendix D.1 p.20
工程啟發

一個好的 medical AI agent，不只是回答問題，而是主動修正自己的資料分佈。

Resp-Agent 把「模型錯在哪」變成下一批 synthetic training examples 的生成條件。這才是 closed-loop agent system 的含金量。

§6 p.10; Appendix B.2 p.18

限制與安全邊界

  • Research only: authors explicitly state it is not a certified medical device and cannot influence patient care without validation and regulatory oversight。
  • Summary quality: clinical summaries come from heterogeneous public metadata, not full EHR；source coverage 會影響文字訊號。
  • Compute cost: some LoSO planner settings are omitted because retraining across five folds is expensive。
  • Scope: evidence covers respiratory audio datasets；generalizing to other medical domains still needs domain-specific generator + evaluation。
Ethics p.11; §3 p.3; Table 3 p.9; §6 p.10

Appendix · 原文螢光筆 1

“Resp-Agent, an autonomous multimodal system orchestrated by a novel Active Adversarial Curriculum Agent (Thinker-A2CA)… actively identifies diagnostic weaknesses and schedules targeted synthesis in a closed loop.”

Resp-Agent 的關鍵不是把幾個模型串起來，而是讓 Thinker-A2CA 根據診斷弱點主動安排合成資料，形成閉環課程。Abstract p.1

“For a 10 s segment embedded into T=496 time steps… adjacent global anchors are spaced by… 80.6452 ms… worst-case deviation ≤40.3 ms.”

這個數字把 audio anchor 從抽象設計落到臨床聲音事件尺度：不用 full quadratic，也能以 sub-100ms 解析度對齊瞬態聲音。§4.2.2 p.8

Appendix · 原文螢光筆 2

“Thinker-A2CA achieves a Macro-F1 of 0.598 and a Macro-F1tail of 0.421… +0.052 Macro-F1 and +0.045 Macro-F1tail over… static-uncertainty baseline.”

同預算比較才看出 agent planner 的價值：不是合成越多越好，而是選對該合成的 rare / hard cases。§5 p.9

“Tightly coupling where the model fails with what the generator synthesizes transforms augmentation from a passive remedy into an active, self-correcting training curriculum.”

把 failure mining 與可控生成器接起來，augmentation 就從資料前處理變成系統級控制迴路。§6 p.10
