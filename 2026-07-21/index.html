Adaptive Auxiliary Prompt Blending for Target-Faithful Diffusion Generation

Kwanyoung Lee, SeungJu Cha, Yebin Ahn, Hyunwoo Oh, Sungho Koh, Dong-Jin Kim · Hanyang University
CVPR 2026 main trackarXiv:2603.19158v1training-free diffusion control

把 auxiliary prompt 從手工 schedule 變成每個 timestep 的 score-space projection。

Source: arXiv abs comment; cover p.1
TL;DR

Rare concepts drift toward high-density modes; AAPB corrects the trajectory by computing a closed-form anchor weight at every denoising step.

核心不是再訓練模型，而是在 inference score space 裡動態平衡 target fidelity 與 anchor stability。

§1 p.1-2; §4 p.3-5
Problem

Low-density concepts make diffusion scores under-constrained

  • Long-tailed text-image data over-represents common concepts and leaves rare combinations sparsely constrained.
  • Denoised estimates drift toward high-density modes: e.g., hairy frog can collapse toward more dominant frog/animal features.
  • The same geometry appears in editing: target guidance can override source structure; source anchoring can suppress the edit.
The failure is a score-geometry problem, not just a prompt-wording problem.
§1 p.1; Fig.1 p.1
Prior methods

Anchor prompts help, but fixed schedules are brittle

  • R2F uses frequent prompts as semantic anchors, but alternates prompts with fixed / heuristic schedules.
  • Prompt-level alternation prevents continuous score-space interpolation and cannot react to the current denoising state.
  • Image editing baselines face a similar fixed trade-off: preserve source structure or follow target instruction, but not both robustly.
AAPB asks: can the anchor weight be solved rather than tuned?
§2 p.2; §5.2 p.7-8
Contribution map

AAPB makes anchor guidance continuous, adaptive, and training-free

  1. Unified task view: rare generation uses a frequent semantic anchor; editing uses the source prompt as anchor.
  2. Closed-form gamma: derive gamma*t from Tweedie / score alignment, no per-prompt grid search.
  3. Theory + mechanism: adaptive projection lowers local score error and tightens an idealized W2 bound.
  4. Evidence: improves RareBench and FlowEdit while keeping the method inference-only.
§1 p.2; §4.2 p.4-5; Tab.1-2 p.5-7
Method · score blending

Step 1: replace CFG’s conditional branch with target-anchor interpolation

s(xt,c) = (1 - gamma_t)s(xt,cT) + gamma_t s(xt,cA)
s_hat = su + w((1 - gamma_t)sT + gamma_t sA - su)
  • cT: rare target concept or edited target prompt.
  • cA: frequent semantic anchor for rare generation; original source prompt for editing.
  • gamma_t: controls how much anchor score is mixed into the conditional branch at timestep t.
§4 Eq.7-8 p.3
Method · Tweedie link

Step 2: posterior mean alignment turns image fidelity into score loss

L(gamma_t) = ||s_hat(xt; w, gamma_t) - sT(xt)||2

Tweedie’s formula makes the denoised posterior mean proportional to the score, so a target-faithful image objective can be optimized in score space.

  • Minimize distance between blended denoiser and target denoiser.
  • The alpha_t pre-factor depends only on the noise schedule, so it does not change the optimal gamma.
  • This formalizes “use the anchor, but stay on the target trajectory.”
§4.1 Eq.9-12 p.4
Method · closed form

Step 3: gamma becomes a projection coefficient, not a knob

gamma*t = ((1 - w) / w) · <sT - su, sA - sT> / ||sA - sT||2
  • Numerator: how much the target residual aligns with the anchor-target direction.
  • Denominator: normalizes by anchor-target displacement magnitude.
  • If the anchor direction is helpful, gamma rises; if it is orthogonal or harmful, its influence is reduced.
§4.2 Eq.13 p.4; App.§8 p.11-12
Algorithm

AAPB drops into sampling as a per-step score update

  1. Sample xT from Gaussian noise.
  2. For each timestep, compute unconditional score su, target score sT, and anchor score sA.
  3. Compute gamma*t from the closed form.
  4. Construct the blended guided score and run the sampler step.
  5. Return x0; no model weights, LoRA, or prompt optimization are trained.
The extra cost is additional score evaluations / blending logic, not training infrastructure.
Algorithm 1 p.13; §5 p.5
Mechanism evidence

Toy Gaussian: adaptive beats every fixed gamma choice

  • Training mixture: 80% auxiliary anchor samples and 20% target-prior samples.
  • Fixed interpolation has a convex W2 curve with a minimum around gamma≈0.8.
  • Adaptive gamma* changes by denoising state and stays below the fixed curve.
The toy setting isolates the central claim: optimal anchor strength is not constant.

Coefficient dynamics

Gamma behaves differently for editing and rare generation

  • Editing: source-image structure dominates later, so auxiliary correction decays / saturates.
  • Rare generation: no source structure exists, so anchor stabilization remains useful throughout denoising; mean gamma is around 0.5.

Mechanism crown

AAPB works when the anchor direction is useful but not trusted blindly.

The coefficient is literally a projection: keep the component that helps align with the target residual, suppress the rest.

§4.2 p.4; App.§14 p.15
Theory

What the W2 result does and does not prove

  • Claim: under log-concavity and transport-information inequality, pointwise projection lowers Fisher divergence versus any fixed scalar gamma.
  • Consequence: lower Fisher divergence gives a tighter squared W2 upper bound to the target distribution.
  • Important caveat: natural images are not globally log-concave.
  • Practical role: theory motivates local score-error minimization; experiments carry the real evidence.
W22(qproj,pT) <= (1/k2)J(qproj||pT) <= (1/k2)J(qgamma||pT)

Limitations

AAPB still cannot fix compositional binding in the encoder

  • Failure mode: multi-attribute / multi-object prompts can still leak attributes to the wrong object.
  • Root inherited issue: CLIP-based text encoders struggle with compositional binding.
  • Possible future fix: representation-level or architecture-level modules for explicit attribute binding.

Also note a slight LAION-aesthetic trade-off: Ours 6.191 vs SD3.0 6.317 on Tab.10.

§21 p.18; Fig.13 p.18; Tab.10 p.18
Engineer’s read

What image-generation engineers can reuse

  1. Score-space controls beat prompt heuristics when the failure is geometric and timestep-dependent.
  2. Anchors should be semantically nearby, not richly detailed; too much orthogonal displacement adds artifacts.
  3. Editing and rare generation can share the same inference module if “anchor” is abstracted cleanly.
  4. Metrics must separate text adherence and structure preservation; CLIP-T alone hides editing damage.
